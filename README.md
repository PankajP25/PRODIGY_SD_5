E-Commerce websites supported:
<ul type='square'>

  <li>Flipkart</li>
  <li>Snapdeal</li>

This web application utilizes:
<ul type='square'>

  <li>HTML and CSS as front end --> <a href='Templates/index.html'>index.html (search page)</a> & <a href='Templates/result.html'>result.html (results page)</a></li>
</ul>
